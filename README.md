# ClassicModels Sales Data Analysis
Excel, Power BI
•	Cleaned and standardized 8 relational datasets (customers, orders, products, employees, etc.) in Excel to ensure data quality before analysis.
•	Built a multi-page interactive Power BI dashboard with DAX measures for Sales, Profit, and Profit Margin %; KPI cards; geographic maps; monthly trend line charts; and top/bottom product bar charts.
•	Implemented slicer-based filtering, cross-visual drill-through, and bookmark navigation to analyze product performance, customer distribution, and employee sales across global operations.
•	Modeled relationships across 7 tables (orders, order details, product, product line, customer, employee, date) enabling dynamic cross-filtering throughout the report.
